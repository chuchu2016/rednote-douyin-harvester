# 隐私政策 (Privacy Policy)

**生效日期：2026年1月6日**

"小红书抖音整理助手"（以下简称"本扩展"）非常重视您的隐私。本隐私政策旨在说明本扩展如何读取、使用和保护您的信息。

## 1. 信息读取与使用

本扩展**不会**将您的个人敏感信息或整理结果发送到开发者服务器。

*   **用户配置信息**：您填写的飞书 App ID、App Secret 等配置信息仅保存在当前浏览器本地 (`chrome.storage.local`)，用于与飞书 API 进行通信。
*   **网页可见信息**：本扩展仅在您主动点击整理按钮时，读取当前抖音/小红书页面已加载、可见的公开信息（作品标题、链接、作者信息和点赞数）。这些信息可由您选择导出为本地 Excel 文件，或保存到您配置的飞书表格中。
*   **Excel 导出**：当您点击导出 Excel 时，文件会在浏览器本地生成并下载，不会发送到开发者服务器。
*   **信息流转**：保存到飞书时，信息流向仅为：您的浏览器 -> 飞书 API。信息不经过任何开发者服务器。

## 2. 权限说明

*   **activeTab**：用于检测您当前是否处于小红书或抖音页面。
*   **storage**：用于在当前浏览器本地保存您的插件配置。
*   **host_permissions**：
    *   `douyin.com`：用于读取视频标题、链接和点赞数。
    *   `xiaohongshu.com`：用于读取作品标题、链接和点赞数。
    *   `open.feishu.cn`：用于将数据保存到您的飞书表格。

## 3. 第三方服务

本扩展的功能依赖于飞书 API。在使用保存功能时，需遵循飞书的相关隐私政策。

## 4. 联系方式

如果您对本隐私政策或合规使用边界有任何疑问，可以通过项目发布页或代码仓库的 Issue 渠道联系维护者。
